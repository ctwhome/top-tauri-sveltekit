# Template Repository : SvelteKit + TailwindCSS + Tauri App
by @ctwhome

```shell
# install dependencies
pnpm i
# run development mode
pnpm taui dev
# build SvelteKit
pnpm build
# build tauri app  - it needs to build the frontend first
pnpm tauri build

# relsese: Build both, frontend and backend
pnpm release
```
